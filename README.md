# 🚀 Blog Management System

Sistema de gestión de contenido desarrollado con **Laravel 10**, diseñado para la administración eficiente de publicaciones, categorías, etiquetas y usuarios mediante una interfaz moderna y escalable.

## ✨ Características

* Gestión completa de publicaciones (CRUD).
* Administración de categorías y etiquetas.
* Sistema de autenticación y autorización.
* Roles y permisos.
* Dashboard administrativo.
* Componentes reactivos con Livewire.
* Interactividad ligera con Alpine.js.
* Arquitectura preparada para producción.

## 🛠️ Tecnologías

* PHP 8.1+
* Laravel 10
* Livewire
* Alpine.js
* Laravel Jetstream
* MySQL / PostgreSQL
* Blade & Vite

## 🏗️ Arquitectura

El proyecto implementa una arquitectura basada en **MVC** y **Service Layer**, promoviendo la separación de responsabilidades, mantenibilidad y escalabilidad del código.

## 🔒 Seguridad

* Autenticación segura con Laravel Jetstream.
* Control de acceso basado en roles y permisos.
* Protección de rutas y recursos sensibles.
* Validación de datos en cliente y servidor.

## 🚀 Instalación

```bash
git clone https://github.com/usuario/proyecto.git

cd proyecto

composer install
npm install

cp .env.example .env

php artisan key:generate

php artisan migrate

php artisan serve
```
